# Yagura (櫓)

> The watchtower over your infrastructure.

Yagura is a unified operational dashboard for self-hosted infrastructure—bringing services, containers, health, metrics, events, and logs into a single elevated view.

Like the watchtower of a Japanese castle, Yagura quietly surveys the entire compound. It stays calm while everything is healthy, and only calls attention when something needs it.

It answers:

* What is running?
* What is unhealthy?
* What changed?
* What needs attention?

Yagura observes and signals.

It does not deploy, orchestrate, or repair.
